# MedBench Frontend

React application built with Vite and TypeScript.

## Setup

Create `.env.local` with required environment variables (see `.env.example`).

```bash
npm install --legacy-peer-deps
```

## Development

Opens development server at [http://localhost:5173](http://localhost:5173).

```bash
npm run dev
```

## Build

Builds production bundle to `build/` directory.

```bash
npm run build
```

## Test

Runs Vitest test suite.

```bash
npm run test
```
