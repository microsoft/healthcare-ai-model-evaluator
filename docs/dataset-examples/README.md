**Dataset extraction and preparation examples**

These example notebooks show how you can download and prepare data for ingestion by Healthcare AI Model Evaluator:
- **example_huggingface_pipeline:** connects to, downloads and prepares several text-based datasets from huggingface, and stores them locally as jsonl files.
- **example_kaggle_pipeline:**  connects to, downloads and prepares a sizeable image-dataset including labels, stores it locally as a jsonl file including base64 encoded images.
